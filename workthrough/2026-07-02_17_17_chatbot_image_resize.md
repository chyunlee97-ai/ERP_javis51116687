# 챗봇 아바타 이미지 크기 조정 및 위치 미세 조정

## 개요
챗봇 아바타 이미지(`ohsung_mark_256.svg`)의 표시 크기를 기존 대비 70% 수준으로 축소하고, 정중앙 정렬을 위해 Y축 좌표를 2픽셀 상향 조정하였습니다.

## 주요 변경사항
- **개선한 것**: `client/ui/scifi_avatar.py`에서 챗봇 이미지 표시 크기를 기존 `133x133`에서 70% 축소된 `93x93`으로 변경
- **개선한 것**: 챗봇 이미지가 동심원 디자인의 중앙에 완벽히 정렬되도록 Y축 좌표를 -2픽셀 이동 (`cy - img_h/2 - 2`)

## 핵심 코드
```python
# client/ui/scifi_avatar.py
            img_w, img_h = 93, 93 # 기존 133 대비 70% 크기 축소
            target_rect = QRectF(cx - img_w/2, cy - img_h/2 - 2, img_w, img_h) # 2픽셀 상향 이동
```

## 결과
- ✅ 크기 축소로 인해 아바타 이미지가 공상과학(Sci-Fi) 동심원 이펙트 내에 더욱 조화롭게 안착함
- ✅ Y축 2픽셀 상향 이동을 통해 오성 로고가 정중앙에 완벽히 정렬됨을 검증 완료
