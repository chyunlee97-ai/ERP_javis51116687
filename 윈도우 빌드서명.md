"현재 개발 중인 파이썬 팝업 챗봇 프로그램을 윈도우용 단일 실행 파일(.exe)로 빌드하고, 이를 사내에 안전하게 배포하기 위한 패키징 작업을 수행해줘. 아래 조건들을 반영해줘:PyInstaller 빌드 설정: 백그라운드 콘솔 창이 뜨지 않도록 -w (Windowed) 옵션을 적용하고, 단일 파일 형태로 만들기 위해 -F (Onefile) 옵션을 사용해줘. 프로그램 실행 시 항상 최상위 권한을 유지하도록 --uac-admin 옵션도 함께 추가해줘.인스톨러(Installer) 패키징 스크립트 생성: AD 미접속자가 이 프로그램을 설치할 때 사내 인증서가 자동으로 심어지도록 Inno Setup 스크립트(.iss)를 작성해줘.인증서 자동 등록 로직 구현: 인스톨러가 실행되는 도중(설치 과정) 배경에서 certutil 명령어를 호출하여 /build_assets/my_company.cer 파일을 시스템의 Root와 TrustedPublisher 영역에 자동으로 등록하도록 스크립트에 반영해줘.최종 결과물로 빌드 자동화 스크립트와 Inno Setup 파일을 아티팩트(Artifacts)로 만들어줘."
